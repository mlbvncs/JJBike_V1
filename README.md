# JJBike PostgreSQL

A Business Intelligence project that combines PostgreSQL and Power BI to analyze and visualize business data from the fictional company JJBike.

## About the project

The purpose of this project is to demonstrate the development of a Business Intelligence solution using PostgreSQL as the database management system and Power BI as the visualization tool.

The project covers data organization, transformation, modeling, and analysis to support data-driven decision-making.

## Technologies used

- PostgreSQL.
- SQL.
- Power BI.
- Git.
- GitHub.

## Project objectives

- Organize and manage data using PostgreSQL.
- Perform SQL-based data extraction and analysis
- Build Key Performance Indicators (KPIs).
- Create interactive dashboards with Power BI.
- Generate insights to support business decisions.

## Analysis highlights

The project may include analyses such as:

- Total sales performance.
- Revenue trends over time.
- Best-selling products.
- Category performance analysis.
- Customer purchasing behavior.
- Profitability indicators.
- Operational performance metrics.

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/mlbvncs/JJBike_PostgreSQL.git
```

### 2. Navigate to the Project Directory

```bash
cd JJBike_PostgreSQL
```

### 3. Set Up PostgreSQL

- Create a PostgreSQL database.
- Import the provided SQL scripts.
- Execute the queries available in the project files.

### 4. Open Power BI

- Connect Power BI to the PostgreSQL database.
- Refresh the data source.
- Explore the dashboards and reports.
